# Sales Intelligence System - Quick Guide

## Installation & Setup

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

That's it! Your sales intelligence system will be running at `http://localhost:3000`

## What's Included

- **Next.js Frontend** - React dashboard with sales analytics
- **Machine Learning** - Sales prediction models (Random Forest, XGBoost)
- **Rossmann Dataset** - Store sales data for training
- **Real-time Predictions** - 30-day sales forecasting

## Features

- 📊 Sales Dashboard with KPI cards
- 📈 Interactive charts and visualizations
- 🤖 ML-powered sales predictions
- 📱 Responsive design
- 🎯 Product and regional analysis

Visit `http://localhost:3000` after running `npm run dev` to see your sales intelligence system in action!
